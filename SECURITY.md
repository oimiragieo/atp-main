# Security Policy

## Supported Versions

This project is under active development. Security updates are provided on a best-effort basis.

## Reporting a Vulnerability

- Please open a private security advisory or contact the maintainers via email listed in the repository.
- Do not create public issues for vulnerabilities.
- Provide reproduction steps, affected versions, and potential impact if known.

## Responsible Disclosure

We appreciate responsible disclosure and will acknowledge contributors who help improve security.
